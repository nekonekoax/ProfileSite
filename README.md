# ねこにゃんこ プロフィールサイト 🐱

ストリーマー「ねこにゃんこ」の自己紹介プロフィールサイトです。

## 🌟 機能

### デザイン
- モダンで可愛らしいグラデーションデザイン
- アニメーション背景（浮遊する肉球エフェクト）
- レスポンシブデザイン（スマートフォン、タブレット、PC対応）
- ガラスモーフィズムUIカード

### アニメーション
- ページロード時のフェードインアニメーション
- ホバーエフェクト（カード、ボタン、ゲームタグ）
- プロフィール画像のパルスアニメーション
- リング拡張エフェクト
- タイトルグロウエフェクト
- ハートビートアニメーション

### インタラクティブ機能
- ソーシャルリンクのリップルエフェクト
- マウストレイルエフェクト（PC版のみ）
- プロフィール画像イースターエッグ（5回クリックで紙吹雪）
- スムーズスクロール
- Intersection Observer による遅延アニメーション

### パフォーマンス最適化
- GPU アクセラレーション
- レイジーローディング対応
- アニメーション軽減モード対応（prefers-reduced-motion）
- バッテリーレベルに応じたアニメーション調整
- タブ非表示時のアニメーション一時停止

## 📱 レスポンシブ対応

- **モバイル（〜480px）**: 1カラムレイアウト、最適化されたフォントサイズ
- **タブレット（481px〜768px）**: 調整されたレイアウト
- **デスクトップ（769px〜）**: 完全なデスクトップエクスペリエンス

## 🚀 使用方法

1. `index.html` をブラウザで開く
2. または、ウェブサーバーにアップロードして公開

### ローカルサーバーで実行（推奨）

```bash
# Python 3 を使用
python -m http.server 8000

# Node.js の http-server を使用
npx http-server
```

その後、ブラウザで `http://localhost:8000` を開きます。

## 🎨 カスタマイズ

### 色の変更
`style.css` の `:root` セクションでカラー変数を変更:

```css
:root {
    --primary-color: #f7c7fa;
    --secondary-color: #5568fc;
    --accent-color: #b794f6;
    /* ... */
}
```

### コンテンツの変更
`index.html` の該当セクションを編集してください。

### アニメーションの調整
`style.css` の `@keyframes` セクションでアニメーション速度や種類を調整できます。
